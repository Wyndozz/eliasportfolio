Parcours

Interressé par le Web, j'ai commençé à vouloir programmer des sites internet. J'ai commençé avec les langages basiques html et css puis j'ai approfondi avec du Javascript. Je maîtrise et sais utiliser plusieurs langages dont le html, css, le Javascript, le C et GIT. Je sais aussi utiliser des frameworks comme bootstrap ou encore jQuery. A l'avenir je souhaite aussi connaitre et savoir le langage php afin d'approfondir mon savoir. Mais ce que j'aimerais le plus apprendre c'est le React car React est une bibliothèque puissante et populaire qui permet de développer des interfaces utilisateur de manière simple et efficace, et qui est facile à intégrer dans des projets existants. React est très populaire et utilisé par de nombreuses grandes entreprises, ce qui signifie qu'il y a une grande communauté de développeurs qui l'utilisent.
